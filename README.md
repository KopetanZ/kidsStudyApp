# 🎓 Kids Study App - 小学生向け学習アプリ

文部科学省学習指導要領に基づいた、小学校低学年向けの包括的学習システムです。ひらがなから漢字まで、基礎から応用まで体系的に学べます。

## 🌟 主要機能

### 📚 学習科目
- **🔢 算数**: 数の理解(1-100) → 繰り上がり・繰り下がり → 四則演算
- **🇯🇵 国語**: ひらがな → カタカナ → 1-2年生漢字 → 単語・熟語
- **🇺🇸 英語**: アルファベット → 基本単語 → フォニックス
- **🕐 時計**: ○時 → ○時30分 → 15分・45分読み
- **🔺 図形**: 基本図形 → 図形比較 → パターン認識

### 🎯 教育的特徴
- **段階的学習設計**: 基礎から応用まで体系的に構成
- **視覚的サポート**: さくらんぼ計算、十進ブロック、筆順ガイド等
- **多感覚学習**: 視覚・聴覚・書字を統合したアプローチ
- **学習指導要領準拠**: 文科省基準に基づく教育内容

### 🎮 ゲーミフィケーション
- **7段階レベルシステム**: しょしんしゃ → だいはかせ
- **バッジ・マイルストーン**: 学習成果の可視化
- **学習進捗トラッキング**: リアルタイム統計とグラフ
- **励ましシステム**: AI生成の個別モチベーションメッセージ

### 🔊 音声機能
- **日本語音声合成**: ひらがな・カタカナ・漢字対応
- **子供向け最適化**: 適切な速度・音高で親しみやすく
- **学習フィードバック**: 正解/不正解時の音声ガイド
- **読み上げボタン**: 全ての学習コンテンツに対応

## 🏗️ 技術構成

### フロントエンド
- **Next.js 15**: React App Router使用
- **TypeScript**: 型安全な開発
- **Tailwind CSS**: レスポンシブUI

### 学習エンジン
- **MathQuestionGenerator**: 算数問題生成（さくらんぼ計算等）
- **JapaneseQuestionGenerator**: 国語問題生成（ひらがな〜漢字）
- **KanjiQuestionGenerator**: 漢字学習システム
- **NumbersQuestionGenerator**: 数概念強化システム
- **TimeQuestionGenerator**: 時計読み問題
- **ShapeQuestionGenerator**: 図形認識問題

### データ管理
- **LocalStorage**: 学習進捗の永続化
- **ProgressTracker**: 包括的学習分析
- **BadgeSystem**: 達成度管理
- **SpeechSynthesis**: 日本語音声読み上げ

## 🚀 セットアップ

### 前提条件
- Node.js 18+
- npm または yarn

### インストール
```bash
# リポジトリクローン
git clone https://github.com/KopetanZ/kidsStudyApp.git
cd kids-study-app

# 依存関係インストール  
npm install

# 開発サーバー起動
npm run dev
```

### ビルド
```bash
# 本番ビルド
npm run build

# ビルド結果確認
npm run start
```

## 🎯 学習システム設計

### 算数学習フロー
1. **数の理解**: 1-10 → 1-20 → 1-100の段階的数概念構築
2. **四則演算基礎**: 視覚サポート付き一桁計算
3. **繰り上がり・繰り下がり**: さくらんぼ計算・借用可視化
4. **多桁計算**: 段階的難易度上昇

### 国語学習フロー  
1. **ひらがな**: あ〜さ行 → た〜は行 → ま〜わ行
2. **ひらがなクイズ**: 画像から文字推測
3. **カタカナ**: 直線重視で漢字準備、外来語学習
4. **漢字**: 1年生基礎 → 2年生学校生活語彙
5. **単語・熟語**: 実用的語彙習得

### 進捗管理システム
- **レベル計算**: ポイント蓄積による自動レベルアップ
- **マイルストーン**: 科目別・総合的達成目標
- **学習統計**: 正答率・学習時間・強み/弱み分析
- **継続促進**: ストリーク管理・日次目標設定

## 📈 学習効果

### 教育的根拠
- **多重感覚学習**: 視覚・聴覚・運動感覚の統合
- **段階的習熟**: スキャフォールディング理論に基づく設計
- **即座フィードバック**: 学習効果を最大化する即時評価
- **個別化学習**: 進捗に応じた適応的難易度調整

### 学習成果指標
- **理解度向上**: 視覚的サポートによる概念理解促進
- **学習継続率**: ゲーミフィケーションによる内発的動機向上
- **基礎学力定着**: 反復学習による自動化達成
- **学習時間最適化**: 効率的な学習パス提供
